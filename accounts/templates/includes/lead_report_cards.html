{# Lead Management report cards — used on /leads/reports/ and /all-reports/ #}
Lead Summary

All leads with status counts for date range.
Status Report

Leads by New, Contacted, Confirmed, Closed, etc.
Source Report

Leads by Call, Website, Social, Referral, etc.
Project Report

Leads grouped by project interest.
Staff Report

Leads assigned to each manager / executive.
Conversion Report

Confirmed vs closed (lost) leads in period.
Follow-up Report

Open leads with follow-up dates in range.
Activity Report

Calls, visits, notes logged in date range.
